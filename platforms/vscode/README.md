# Unix Utility Suite Launcher

Launches the installed `unix-toolkit` Linux desktop utility from Visual Studio
Code.

## Requirements

Install Unix Utility Suite first:

```bash
sudo apt install ./unix-utility-suite_1.1.0_amd64.deb
```

Then run the command:

```text
Unix Utility Suite: Launch Toolkit
```

If the command is installed somewhere other than `unix-toolkit`, set
`unixUtilitySuite.executablePath` in VS Code settings.
