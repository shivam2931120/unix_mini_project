# Changelog

## 1.1.0 - 2026-05-31

- Added direct `unix-toolkit` command-line actions for GNOME Shell menu integration.
- Added GNOME Shell extension preferences with menu-action toggles.
- Added GNOME Shell install detection with GitHub release fallback actions.
- Added a GNOME Shell keyboard shortcut for opening the full toolkit launcher.

## 1.0.0 - 2026-05-21

- Added installable Linux command layout for `unix-toolkit`.
- Added Debian package build support.
- Added GNOME Shell and VS Code launcher wrapper projects.
- Added experimental Flatpak launcher manifest.
- Added Docker build file for reproducible build/test checks.
- Hardened scheduler input validation and output handling.
- Reworked process manager and calculator helpers for safer execution.
